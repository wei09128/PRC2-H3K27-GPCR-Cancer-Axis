# 🧬 PRC2-H3K27-GPCR-Cancer-Axis
## A Shared PRC2/H3K27–GPCR Axis Underlies Aggressive Microenvironments in Breast, Ovarian, and Endometrial Cancer

---

## 🎯 Project Summary / Abstract

Aggressive breast, ovarian, and endometrial cancers frequently recur despite targeted and immune therapies, suggesting shared, **microenvironmentally encoded resistance programs**. We assembled a pan-gynecologic, single-cell multi-omic atlas integrating scRNA-seq, scATAC-seq, and H3K27me3 profiles to map epithelial, stromal, and immune states across patient-level response strata, identified a convergent ecosystem characterized by **PRC2/H3K27-linked repression of luminal and antigen-presentation programs in epithelium**, coupled to **GPCR- and matrisome-rewired fibroblast, myeloid, and endothelial compartments**. Cancer type–specific logics (ER⁺/luminal bias in breast, BRCA1/ATM-centered repair in ovarian, TNF/IKK-aligned inflammation in endometrial cancer) instantiate this shared axis while preserving a common pattern of cytotoxic T/NK dysfunction and Treg amplification. Patient-level PRC2- and GPCR-centered modules stratify clinical aggressiveness, nominating an epigenetically restrained, **GPCR-encoded tumor–stroma–immune circuit** as a candidate bottleneck for combinatorial therapeutic targeting across gynecologic malignancies.

## 📁 Repository Structure

This repository is organized to ensure the traceability and reproducibility of the analysis.

| Folder | Content | Notes |
| :--- | :--- | :--- |
| **`code/`** | All **R and Python scripts** used for data processing, differential expression, gene set enrichment, and figure generation. | See **Analysis Workflow** section for run order. |
| **`data/`** | GSE164716, GSE173682, GSE243526 | *Raw data 50 scRNA, 31 scATAC, and 13 scChi samples.* |
